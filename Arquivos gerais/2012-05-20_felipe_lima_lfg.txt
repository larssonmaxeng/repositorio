Todavia, a simples adaptação de uma solução já existente, a qual utiliza o processo de orçamentação tradicional (CAD/Excel/Planilha orçamentária) subutiliza os benefícios proporcionados pelo BIM, pois:
1. Depende da criação de arquivos de texto para troca de dados entre o modelo BIM e o software utilizado para orçamentação, que é um trabalho manual, sujeito a erros que não representa o conceito do BIM;
2. Depende da integração entre o modelo BIM e o software de orçamentação, que não interage de forma eficaz e nem representa o conceito da tecnologia BIM;
3. Não há representação visual do orçamento, no modelo 3D, requisito para ser um software em BIM; e,
4. Deste modo, são geralmente voltados para o uso de software BIM específico (somente para o Autodesk Revit® ou para o Archicad®, por exemplo).
Diante do exposto, propõe-se a criação de uma solução tecnológica para integrar de forma nativa um modelo BIM para o orçamento da obra, eliminando totalmente ou parcialmente as deficiências que podem incorrer em erros de planejamento, evitando a subestimação ou superestimação do custo da obra, evitando que ocorra prejuízos para os envolvidos com uma obra e para toda a sociedade.
